# 数据结构与算法中有那些奇技淫巧？

### 1、用 n & (n - 1)消去 n 最后的一位 1



**判断一个正整数 n 是否为 2 的幂次方**

如果一个数是 2 的幂次方，意味着 n 的二进制表示中，只有一个位 是1，其他都是0

2^0 = 0.....0001

2^1 = 0.....0010

2^2 = 0....0100

2^3 = 0..01000



```java
boolean judege(int n){
return n & (n - 1) == 0;// 
}
```
