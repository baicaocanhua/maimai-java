

# **钉钉内网穿透工具**

git 地址：https://github.com/open-dingtalk/pierced

## windows使用

执行 ding.exe -config=ding.cfg -subdomain=maimai 8080 命令。其中ding.cfg为配置文件所在目录，tuofan为需要的子域名，自定义即可，8896为应用程序所在的端口

## mac使用

```
./ding -config=./ding.cfg -subdomain=maimai 8080
```



## linux使用

```
./ding -config=./ding.cfg -subdomain=maimai 8080
```
